# ورد الشوق — Flowers Longing

موقع تعريفي/تسويقي فاخر لمحل ورد وباقات وهدايا في **حي الصحافة بمدينة الرياض**.

## نظرة عامة
- **الاسم:** ورد الشوق (Flowers Longing)
- **التصنيف:** محل ورد وباقات وهدايا (Florist)
- **المدينة:** الرياض — حي الصحافة
- **التقييم:** 4.7 على خرائط قوقل (459 تقييم) — مُوثّق في الموقع وفي بيانات JSON-LD.

## لماذا اختير هذا النشاط
محل بتقييم عالٍ (4.7) وحجم تقييمات كبير (459)، ونشاط بصري بطبيعته (الورد) يناسب موقعًا أنيقًا يبرز جمال الباقات ويحوّل الزائر إلى طلب عبر واتساب.

## فكرة التصميم
هوية **رومانسية راقية** تميّزه عن أي نسخة "نباتية/سيج" مجاورة:
- لوحة ألوان: **وردي مغبّر + برقوقي غامق + كريمي دافئ + ذهبي ناعم**.
- خطوط: عنوان **El Messiri** (أناقة عربية) + نص **Tajawal** (وضوح).
- موشن لطيف يخدم النشاط: بتلات ورد تنساب في الهيرو، تكبير Ken-Burns بطيء للصورة، ظهور الكروت بأسلوب "تفتُّح" عند التمرير، وتكبير الصور عند المرور.

## التقنية
- **HTML/CSS/JS عادي (Vanilla)** بدون أي خطوة بناء، بدون `node_modules` أو `package.json`.
- خطوط Google عبر `preconnect` + `display=swap`.
- أيقونات SVG inline (بدون إيموجي كأيقونات بنية).

## الرابط الحي

## حالة الـ Backend
**لا يوجد backend / لا نظام دفع.** نموذج الطلب تجريبي:
1. يتحقق من الحقول الأساسية (الاسم، الجوال، نوع الطلب).
2. يحفظ نسخة من الطلب في **localStorage** داخل متصفح الزائر فقط (للعرض).
3. يفتح محادثة **واتساب** مُعبّأة مسبقًا بتفاصيل الطلب على الرقم `966561112769`.
4. يعرض **Toast** نجاح.

## مصدر الصور
صور حقيقية من ملف النشاط على **خرائط قوقل**، تمت **مراجعتها وتنقيتها يدويًا**:
- المُعتمدة: `fl-13512.jpg` (باقة وردي + هدية — الهيرو)، `fl-14696.jpg` (تنسيق أبيض)، `fl-24147.jpg` (تشكيلة المحل).
- المُستبعدة: صور لافتات بأسماء محلات أخرى (ورد الرياض/ورد الشوق على واجهة مختلفة)، وصور ضبابية عبر زجاج السيارة، ولافتات منافسين.
